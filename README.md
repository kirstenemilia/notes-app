# Notes App (Full-stack)

A simple full-stack notes application built with Flask and a Tkinter frontend.  
Allows users to create, view, update, and delete notes stored in a SQLite database.

Features
- Create notes with title and content
- View all saved notes
- Search notes
- Update existing notes
- Delete notes
- REST API backend using Flask
- SQLite database with SQLAlchemy
- Desktop GUI using Tkinter

Built with
- Python
- Flask
- Flask-RESTful
- SQLAlchemy
- SQLite
- Tkinter
- ttkbootstrap

# Project Structure
-backend/ Flask API + database models
-frontend/ Tkinter GUI
-instance/ SQLite database file
-archive/ Old code
